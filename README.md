# manasvi.dev · Portfolio

Personal portfolio of Manasvi Saxena — Full-Stack Developer & Health-Tech builder.

**Stack:** React · CSS (no component library) · deployed on Vercel

## Local dev

```bash
npm install
npm start
```

## Deploy

Linked to Vercel. Any push to `main` triggers a redeploy automatically.

```bash
git add .
git commit -m "update"
git push origin main
```

## Structure

```
src/
  components/   Cursor, Navbar, Marquee
  sections/     Hero, About, Projects, Skills, Contact
  index.css     CSS variables + global styles
```
